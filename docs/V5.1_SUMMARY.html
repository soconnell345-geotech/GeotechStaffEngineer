Behavior changes to know about when testing

  • Seismic passive pressure on battered walls (KPE) changed — pre-5.1 values were from a sign-flipped Mononobe-Okabe convention and were wrong; verified against a numerical wedge solution. Vertical-wall statics unchanged.
  • Pile group: +Mx now consistently uplifts the +y side (right-hand convention end-to-end); was self-inconsistent before.
  • Sheet pile cantilever: single safety basis — design embedment is now 1/1.2× the pre-5.1 value (the old default double-stacked FOS and a 1.2 multiplier); old arithmetic reachable via embedment_increase.
  • Wave equation: default damping is now mobilized-static (GRLWEAP standard); blow counts shift ~2% lower; damping_model="smith_viscous" reproduces v5.0 exactly.
  • fem2d: T6 elements are now the default for collapse/SRM analyses; CST is retained for elastic/seepage/consolidation only.
