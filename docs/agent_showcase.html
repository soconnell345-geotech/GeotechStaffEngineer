How the Agent Works

1

Discover

The LLM browses available methods across all 38 agents by category to find the right function.

list_methods("bearing_capacity") // Returns: bearing_capacity_analysis, // bearing_capacity_factors, ... list_methods("axial_pile", "nordlund") // Returns: nordlund_pile_capacity
2

Inspect

Gets full parameter documentation — types, ranges, defaults, units, and related methods.

describe_method( "bearing_capacity", "bearing_capacity_analysis" ) // parameters, typical_workflow, // common_mistakes, related []
3

Execute

Calls the function with JSON parameters and gets a structured engineering result.

call_agent( "bearing_capacity", "bearing_capacity_analysis", {"B": 2.0, "L": 3.0, "D": 1.5, "phi": 30, ...} ) // q_ultimate_kPa: 1284.5

Example: Multi-Agent Pile Foundation Design

Agent Conversation — Autonomous Tool Selection Across Multiple Modules
User
I have a 16-inch steel pipe pile driven 18 m into medium-dense sand (phi=32, gamma=18.5 kN/m3). The site is in Seismic Site Class D with PGA=0.35g. Check the axial capacity, lateral response to a 90 kN load, and whether the sand might liquefy.
Tool 1
call_agent("axial_pile", "axial_pile_capacity", {"method": "nordlund", "pile_type": "pipe", ...})
{ "Q_ultimate_kN": 2847, "Q_skin_kN": 1923, "Q_tip_kN": 924, "Q_allowable_kN": 1139 }
Tool 2
call_agent("lateral_pile", "lateral_pile_analysis", {"py_model": "SandAPI", "H_kN": 90, ...})
{ "y_top_mm": 8.3, "M_max_kNm": 214, "depth_max_moment_m": 4.2 }
Tool 3
call_agent("seismic_geotech", "simplified_liquefaction", {"N160": 18, "M": 7.5, "PGA": 0.35, ...})
{ "min_FOS_liq": 1.24, "liquefiable": false }
Agent
Summary: The 16" pipe pile has an ultimate axial capacity of 2,847 kN (allowable 1,139 kN at FOS=2.5). Under the 90 kN lateral load, the top deflection is 8.3 mm with maximum moment of 214 kN-m at 4.2 m depth. The site sand has a liquefaction FOS of 1.24 — marginally safe but I recommend checking with CPT-based triggering for confirmation. All three analyses used peer-reviewed, code-compliant methods.
