**B2 round 1 (DONE 2026-07-06, agent `slope-valid`):**
- [x] **SS-6 weak-layer / noncircular search degenerate-surface fix** (`search.py`).
      Root cause: on jagged random polylines the rigorous GLE fails to converge (or
      converges to garbage) and the legacy Spencer fallback returns a spurious low FOS
      (~0.05-0.2) that the search picked as critical (ACADS 4). Fix in `_compute_fos`:
      (1) geometric admissibility (min slices, min span); (2) REQUIRE GLE convergence for
      noncircular trials (`_rigorous_noncircular_fos` → None otherwise, no legacy fallback);
      (3) a low-FOS jaggedness gate (`_is_jagged` + `_LOW_FOS_JAGGED_GATE`) — a low FOS on a
      non-concave surface is a solver artifact. Scoped to low-FOS surfaces so exploration is
      untouched; circular path byte-identical. Re-validated #9 (V-028 → PASS). Slope suite
      green (test_b2_round1.py adds 9 tests). Commit: <search-fix>.
